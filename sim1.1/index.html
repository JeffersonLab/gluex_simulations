Output File Location

Job Output Files

The output files are located in /volatile/halld/gluex_simulations/sim1.1 There are directories there, that contain output files for each job. Each output file type has its own directory:
  • hd_root: ROOT files from hd_root
  • rest: REST format event data
  • smeared: HDDM formated "raw" event data (without reconstruction)
  • tree_bcal_hadronic_eff: ROOT files from the plugin
  • tree_fcal_hadronic_eff: ROOT files from the plugin
  • tree_sc_eff: ROOT files from the plugin
  • tree_tof_eff: ROOT files from the plugin
Use the "jcache" or "jget" commands to retrieve the files from tape.

Job Log Files

Standard output and standard error files from each job are in /work/halld/gluex_simulations/sim1.1/logs. The run number and file number are encoded in the name. Note that a particular run-number/file-number may correspond to multiple jobs; you are probably interested in the latest one.
